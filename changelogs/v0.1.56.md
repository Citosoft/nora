### Features

- Review changes inline in the diff viewer: add notes on changed lines, manage them in a review tray in the Changes panel, and send batched feedback back to a running agent.
- Opening a diff for the first time shows a short guide to inline notes and the review tray.
- Agent settings can install missing agent tools for you and show live install progress.
- The agent catalog includes more built-in agents, each with install instructions.
- Settings include an option to run onboarding again.
- Help includes a Resource Monitor with live CPU and memory graphs, plus a breakdown of app, agent, and terminal processes.
- Commit message generation can run on a local model instead of a cloud API key—choose on-device in Settings → AI and install the runtime and model from there.
- Voice input supports cloud transcription or on-device Whisper, with one-click model setup, clearer listening and transcribing states, a level meter while recording, and a stop button on the mic.

### Improvements

- Dictation no longer shows an error when you stop recording without speaking.
- Onboarding is easier to follow: clearer progress and step titles, optional vs required setup cards, copyable install commands, theme previews, and a dedicated appearance step.
- Tooltips are easier to read in dark mode, size to their content, and appear above dialogs instead of behind them.
- Diff review comments and changed-file rows are more compact in the Changes panel.
- Inline diff comments use clearer highlighting so they stand out in the diff.
- More agent tools are detected automatically after you install them.
- Workspace sidebar section counts appear inline in section headers.

### Fixes

- Center tab Close Others now closes agent and terminal tabs correctly, with one confirmation when closing multiple agents.
- The macOS Help menu shows Resource Monitor right away, even before the rest of the menu finishes loading.
- New Agent dialog: the Start button looks correct in light and dark mode, and the task selector defaults to No task.
- Terminal tabs remember and restore the working directory for each tab.
- Diff review notes are saved per workspace and survive restarts.
- Diff layout stays tight without extra spacing from inline notes.
- The footer shows only agents you have enabled and installed; usage and account controls appear only when the agent supports them.
- Account switch errors use the standard error dialog; usage refreshes without repeated sign-in prompts when an agent is signed out; Login appears when you are not authenticated; Claude Code sign-in no longer leaves the switch button stuck while login is pending.
- Agent icons use official logos when available, with good contrast in dark mode.
- Newer catalog agents show proper icons and install buttons in settings.
- Worktree menus correctly enable and disable actions and run workspace scripts from worktrees.
